# Firefox Extension Settings

After logging into Firefox Sync (which restores extensions), restore each
extension's settings from the backup files in this directory.

## uBlock Origin

**Export:** Dashboard (click uBlock icon → gear) → Settings tab → "Backup to file"
**Import:** Dashboard → Settings tab → "Restore from file..." → select `ublock-settings.json`

## Global Speed

**Export:** Extension popup → gear icon → Export
**Import:** Extension popup → gear icon → Import → select `global-speed-settings.json`
